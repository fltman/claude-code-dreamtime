# Dreamtime

[![Support me on Patreon](https://img.shields.io/badge/Patreon-Support%20my%20work-FF424D?style=flat&logo=patreon&logoColor=white)](https://www.patreon.com/AndersBjarby)

DREAMTIME is a Claude Code extension that treats context compression as knowledge transformation rather than information loss. When an agent's context fills and compresses, DREAMTIME frames it as "dreaming": a deliberate process that distills raw experience into insights, warnings, narrative songlines, and shadows that mark where knowledge was lost.

It builds on the FLUX / METAMORPHOSIS / SPORE stack, adding a consolidation layer on top of evolution, governance, and peer-to-peer knowledge sharing.

## What it does

- **Deliberate dreaming** — agents enter a dream state with an intention and depth, turning accumulated experience into artifacts
- **Insights** — pattern discoveries surfaced through compression
- **Warnings** — inarticulate risk signals that get a voice
- **Songlines** — narrative knowledge paths that transfer wisdom across contexts
- **Shadows** — markers that record where knowledge was lost, so forgetting becomes a signal rather than silence

## Commands

- `/dream` — enter dream state with intention and depth
- `/orient` — wake from a dream and integrate artifacts
- `/shadows` — review active shadows and their triggers
- `/songlines` — browse and search songlines
- `/interpret` — interpret a foreign songline for local context

## Agents

- **dreamer** — manages dream entry, compression, and artifact generation
- **interpreter** — translates foreign songlines into local context

## Usage

This is a Claude Code project. Open it with Claude Code and use the slash commands above; the dream cycle is Gather → Dream → Orient → Resume, repeated to deepen understanding over time.

```bash
claude
/dream
```

## Tech

Claude Code (CLAUDE.md instructions, `.claude/` agents, commands, and spore-type definitions). Markdown-based; no build step.
